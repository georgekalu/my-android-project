# 🚀 Acode Mini Apps Hub

A collection of fun and useful Python mini applications built entirely inside **Acode** on Android.

## Apps Included

- **Calculator** – Enhanced calculator with power & loop
- **Guessing Game** – Guess the secret number (1-100)
- **To-Do List** – Persistent task manager
- **Quiz Game** – General knowledge quiz
- **Main Menu** – Beautiful colored hub using `rich`
